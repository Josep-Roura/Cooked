# Tokens de diseño

## Colores principales
- `bg` → fondo general de la app
- `surface` → tarjetas, modals
- `border` → líneas divisorias y bordes
- `text.primary` → texto principal
- `text.secondary` → texto secundario / menos importante
- `primary` → CTA, botones principales
- `error`, `success`, `warning` → feedback de estado

## Tipografía
- Fuente base: Inter, system-ui, sans-serif
- Títulos: font-semibold
- Body: font-normal
- Líneas:
  - Título: leading-tight (1.2 aprox)
  - Texto: leading-relaxed (1.5 aprox)

## Radius
- `rounded-lg = 1rem` (cards, modals, botones grandes)
- `rounded-full` para chips/badges

## Sombras
- `shadow-md` en cards
- `shadow-lg` en drawers/modals

## Spacing
- Padding interno de card: `p-6`
- Gap vertical entre secciones dentro de página: `space-y-6`
- Layout page padding: `p-4` en contenido app
